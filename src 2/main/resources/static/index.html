Cassandra CSV Loader · веб

Загрузка CSV в Apache Cassandra через браузер (по умолчанию порт 8090). Десктопная версия — каталог v.0.1.

Подключения к кластерам

Выберите подключение в списке. Добавить / изменить / удалить — кнопками слева.

    Не подключено

    После подключения откройте «Топология кластера» и сверьте Local DC с полем datacenter у узла.
